Swarm Routing Demo

This site is intentionally simple. It exists to prove routing rules, not to win design awards.

Home (file) About (file) Contact (file) /about (clean URL) /about/ (clean URL)

What this demonstrates

  • Exact file paths work immediately: /about.html, /contact.html
  • Clean URLs like /about fail until you add manifest aliases
  • 404 behavior: hitting a missing file should return 404.html

Try these links

These will work right away (no manifest edits):

  • /about.html
  • /contact.html

These will only work after you add aliases like about → about.html:

  • /about
  • /about/

This should trigger your errorDocument:

  • /this-page-does-not-exist

Redirect test

If you later “redirect” /old-page.html to new content by overwriting its manifest entry, you can point it to:

  • /new-page.html

(No HTTP redirects on Swarm—this is “redirect-like” behavior via manifest remapping.)
