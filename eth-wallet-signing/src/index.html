Eth Wallet Signing of Single Owner Chunk

This example writes data to Bee which requires valid Postage Stamp. If you have once, you can pass its Postage Batch ID, if you don't you can create one. But be aware that this creates on-chain transaction and spent Eth (for gas) and BZZ (for the postage stamp). If you are running this against main-net node this will spend real money.

Open the browser console to see what is exactly happening! ;-)
