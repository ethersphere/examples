Why this page exists

This file is named about.html so that “clean URL” routing can be demonstrated by mapping /about and /about/ to it in the manifest.

Without manifest edits, /about is treated as a literal file called about (no extension), which won’t exist.

What you’ll map in the manifest

  • about → reference for about.html
  • about/ → same reference
Tip: open DevTools → Network and watch which file Swarm actually serves.
